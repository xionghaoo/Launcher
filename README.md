# Launcher
> 类似小米平板桌面，基于OpenLauncher修改

特点:

1. 拖动改变图标位置
2. 拖动新增页面，删除页面
3. 图标合并
   + APP -> APP
   + APP -> 文件夹
   + 文件夹 -> APP
   + 文件夹 -> 文件夹
4. 图标文件夹
5. 拖拽卸载App，App安装/卸载监听
